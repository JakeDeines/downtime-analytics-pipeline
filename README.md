# Downtime Analytics Pipeline

Automated downtime data pipeline using Power Automate, SharePoint Lists, and Excel to feed Fabriq dashboards for operational performance tracking.

---

## Overview

This project demonstrates an end-to-end operational data workflow designed to collect, transform, and prepare downtime data for reporting and dashboard visualization.

The system automates the extraction of downtime records, applies filtering and transformation logic, and outputs structured data for use in Fabriq dashboards.

---

## Architecture

Power Automate (ETL Workflow)
→ SharePoint Lists (Source Data)
→ Excel Table (Staging / Structured Output)
→ Fabriq (Dashboard & KPI Visualization)

---

## Key Features

- Automated data extraction using Power Automate
- Filtering and transformation of downtime records
- Structured export to Excel for reporting pipelines
- Integration-ready output for Fabriq dashboards
- Supports KPI tracking and operational performance monitoring

---

## Technologies Used

- Power Automate
- SharePoint Lists
- Excel (Structured Tables)
- Fabriq (Dashboard Platform)

---

## Notes

This repository contains a simplified and sanitized representation of an operational data pipeline. No proprietary or sensitive data is included.
